# Diagram Placeholders

This folder is reserved for public-facing diagrams used in repository documentation.

Suggested future assets:

- system-overview.png
- workflow-lifecycle.png
- data-flow.png

Diagram rules:

- Keep diagrams high-level and public-safe.
- Do not include spreadsheet IDs, folder IDs, deployment URLs, or private contacts.
- Prefer workflow and architecture explanation over implementation trivia.
- Update README links if diagram filenames change.
