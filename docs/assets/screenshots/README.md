# Screenshot Placeholders

This folder is reserved for public-safe screenshots used in the GitHub showcase.

Recommended file names:

- dt-submit-flow.png
- special-request-flow.png
- status-lookup.png
- machine-guide.png
- reviewer-queue.png
- review-drawer.png
- help-guidance.png

Screenshot rules:

- Use the sanitized public branch UI.
- Do not show real names, emails, classes, or IDs.
- Avoid exposing spreadsheet URLs, Drive URLs, or admin-only links.
- Keep the capture style consistent across the set.
- Refresh the README if filenames or screenshot coverage changes.
